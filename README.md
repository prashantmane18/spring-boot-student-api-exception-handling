# Spring Boot Student API with Exception Handling

This project is a RESTful API built using Spring Boot to manage student data.  
It demonstrates CRUD operations and custom exception handling using a clean layered architecture.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 / MySQL
- Maven

## Features
- Create Student
- Get All Students
- Get Student By ID
- Update Student
- Delete Student
- Custom Exception Handling

## API Endpoints

GET /students  
GET /students/{id}  
POST /students  
PUT /students/{id}  
DELETE /students/{id}

## Project Structure

controller  
service  
repository  
entity  
exception  

## Key Concepts Learned
- REST API design
- Layered architecture
- Custom exceptions in Spring Boot
- Service-based business logic
